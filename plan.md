# Final Project – Midterm Plan

## Deploy Link

<https://ntu-exchange-school.vercel.app>

## 1. 本次 Prototype 已完成

- **基本頁面架構**
  - React + TypeScript + Vite 專案架構
  - 響應式佈局設計
  - 基礎 UI 組件系統（使用 shadcn/ui）

- **雙視圖模式**
  - 地圖視圖：使用 React Map GL 顯示所有交換學校位置，支援縮放和標記點擊
  - 表格視圖：以表格形式展示學校詳細資訊，支援排序

- **篩選功能**
  - 地區篩選（美洲、歐洲、亞洲、大洋洲、非洲）
  - GPA 要求篩選
  - 語言能力篩選（TOEFL、IELTS、TOEIC）
  - 學院限制篩選
  - 學期篩選（Fall/Spring）
  - 用戶資格自動匹配功能

- **收藏與比較功能**
  - 收藏感興趣的學校
  - 學校比較功能（CompareModal）
  - 收藏列表管理（WishlistPanel）

- **申請管理框架**
  - 申請表單組件（ApplicationForm）
  - 申請摘要顯示（ApplicationSummary）

- **資料處理**
  - CSV 資料載入與解析
  - 學校地理座標顯示
  - 資料爬取工具（Python scraper）

- **使用者體驗**
  - 歡迎對話框（WelcomeDialog）
  - 浮動搜尋欄（FloatingSearchBar）
  - 面板管理系統（可收合/展開的側邊面板）
  - 動畫效果（Framer Motion）

## 2. 最終版本預計完成項目

### 2.1 核心功能擴充

- **學校資料擴充**
  - 新增篩選指標：
    - 生活機能評分（基於總店家數分布，分 4-5 等級）
    - 物價水準評分
    - 課程 loading 評分
  - 資料整理與爬取：
    - 注意事項整理
    - 實用資訊彙整
    - 學校年曆爬取與整合

- **使用者評分系統**
  - 使用者分享心得時可對學校進行評分
  - 評分項目：生活機能、物價水準、課程 loading、整體體驗
  - 評分資料統計與顯示

### 2.2 社群功能（新增頁面）

- **心得文章分享系統**
  - 部落格/文章發布功能
  - 照片上傳與展示
  - 文章收藏功能
  - 按愛心（Like）功能
  - 留言互動系統
  - 文章與學校標籤關聯

- **地區版功能**
  - 依大洲分類的地區版
  - 顯示該地區所有文章
  - 顯示該地區學校列表
  - 點擊學校可查看所有提及該學校的貼文
  - 支援國家與學校篩選

- **學校討論功能**
  - 發文時可選擇相關學校（0:N 關聯）
  - 下拉式選單選擇該區域的學校
  - 學校專屬討論串

- **閒聊區**
  - 自由發文功能
  - 不限主題的交流空間

### 2.3 側邊欄功能（左側）

- **瀏覽學校**
  - 快速導航至學校瀏覽頁面

- **社群**
  - 進入社群頁面

- **地區版**
  - 快速切換至各地區版
  - 顯示該地區文章與學校

- **學校討論**
  - 查看所有學校相關討論

- **通知系統**
  - 在頭像旁邊顯示鈴鐺圖示
  - 通知小視窗（參考 Facebook 電腦版設計）
  - 通知類型：留言回覆、按愛心、新追蹤者等

- **收藏學校**
  - 快速存取收藏的學校列表

- **我的志願**
  - 選擇志願學校
  - 調整志願順序功能
  - 志願管理介面

### 2.4 使用者認證與社交

- **OAuth 登入**
  - 整合第三方登入（Google、Github 等）
  - 使用者帳號管理
  - 個人資料頁面

- **追蹤功能**
  - 追蹤其他使用者
  - 追蹤列表管理
  - 動態牆顯示追蹤使用者內容
  - 根據追蹤功能調整側邊欄配置

### 2.5 資料庫串接

- **後端 API 開發**
  - 使用者資料管理
  - 文章/貼文 CRUD
  - 留言系統 API
  - 收藏與追蹤功能 API
  - 評分資料儲存與統計

- **資料庫設計**
  - 使用者表
  - 學校表（擴充欄位）
  - 文章表
  - 留言表
  - 收藏表
  - 評分表
  - 追蹤關係表
