## 檔案說明

### 🗄️ 核心資料庫檔案

1. **supabase/schema.sql**
   - 用途：建立資料表、索引、RLS 政策
   - 何時使用：Supabase 專案建立後執行一次
   - 大小：~3 KB

2. **scripts/import-to-supabase.ts**
   - 用途：將 CSV 資料匯入 Supabase
   - 何時使用：資料更新時執行
   - 執行方式：npm run import-data

3. **scripts/test-supabase-connection.ts**
   - 用途：測試 Supabase 連線和資料完整性
   - 何時使用：設定完成後、除錯時
   - 執行方式：npm run test-db

### 🔧 前端整合檔案

4. **src/lib/supabase.ts**
   - 用途：初始化 Supabase 客戶端
   - 匯入：import { supabase } from '@/lib/supabase'
   - 大小：~500 bytes

5. **src/types/supabase.ts**
   - 用途：TypeScript 型別定義
   - 匯入：import type { Database } from '@/types/supabase'
   - 自動產生：未來可用 Supabase CLI 自動產生

6. **src/contexts/SchoolContext.supabase.tsx**
   - 用途：使用 Supabase 的 React Context
   - 切換方式：重新命名為 SchoolContext.tsx
   - 相容性：100% 與現有介面相容

### 📚 文件檔案

7. **docs/QUICK_START.md**
   - 目標讀者：想快速開始的人
   - 預估時間：5 分鐘
   - 內容：精簡的步驟指南

8. **docs/SUPABASE_SETUP.md**
   - 目標讀者：第一次設定的人
   - 預估時間：15 分鐘
   - 內容：完整的教學、常見問題

9. **SUPABASE_MIGRATION.md**
   - 目標讀者：開發團隊
   - 內容：技術細節、架構說明、擴充建議

10. **SUPABASE_README.md**
    - 目標讀者：所有人
    - 內容：總覽、快速參考、檢查清單

### ⚙️ 設定檔

11. **.env.example**
    - 新增欄位：
      - VITE_SUPABASE_URL
      - VITE_SUPABASE_ANON_KEY
      - SUPABASE_SERVICE_ROLE_KEY
    - 注意：記得複製為 .env 並填入真實值

12. **package.json**
    - 新增腳本：
      - npm run import-data
      - npm run test-db

## 使用流程

```
1. 安裝套件
   npm install @supabase/supabase-js
   npm install -D tsx @types/node
   
2. 建立 Supabase 專案
   https://supabase.com/dashboard
   
3. 執行 SQL
   複製 supabase/schema.sql 到 SQL Editor
   
4. 設定環境變數
   cp .env.example .env
   # 編輯 .env 填入金鑰
   
5. 測試連線
   npm run test-db
   
6. 匯入資料
   npm run import-data
   
7. 切換前端
   mv src/contexts/SchoolContext.tsx src/contexts/SchoolContext.csv.tsx
   mv src/contexts/SchoolContext.supabase.tsx src/contexts/SchoolContext.tsx
   
8. 啟動開發
   npm run dev
```
